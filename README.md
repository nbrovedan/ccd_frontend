# Frontend
1. Baixar e instalar o Node.js/npm
```
https://nodejs.org/en/download/
```
2. Instalar o bower
```
npm install -g bower
```
3. Clonar o repositório
```
https://github.com/nbrovedan/ccd_frontend.git
```
4. Acessar o repositório baixado e instalar as dependências:
```
npm install
bower install
```
5. Iniciar o projeto
```
grunt serve
```
